# ShipmentSure ‚Äì On-Time Delivery Prediction App 

This project predicts whether a shipment will be delivered **on time** using Machine Learning.  
A Streamlit web application is built where users can enter shipment details, and the system outputs the likelihood of on-time delivery.

---

##  Features 
- User-friendly Streamlit Web App
- Logistic Regression / Classification model
- One-Hot Encoding for categorical variables
- Standardization using Scaler
- Probability-based prediction with risk levels:
  -  High chance of On-Time Delivery   
  -  Moderate Risk   
  -  High chance of Delay
